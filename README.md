# SIGFA - Sistema de Gestión Farmacéutica

**SIGFA** es una solución integral de software diseñada para la optimización, control y trazabilidad de insumos médicos y medicamentos en entornos hospitalarios. Desarrollado originalmente para la Farmacia del Hospital General Municipal Dr. Juan Daza Pereira en Barquisimeto, Venezuela.

Este sistema ha sido concebido bajo los lineamientos del **Proyecto Socio Tecnológico II** del Programa Nacional de Formación en Informática (PNFI - UPTAEB).

## 🚀 Características Principales

* **Gestión de Inventario Inteligente:** Control estricto de lotes con metodología **FIFO (First-In, First-Out)** para garantizar el despacho de productos según su fecha de vencimiento.
* **Seguridad y Roles:** Autenticación multi-nivel para Administradores, Auxiliares (Farmacia/Alto Costo), Almacenistas y Kardistas.
* **Trazabilidad Total:** Registro detallado de movimientos (Kardex) para auditorías internas y control de gestión.
* **Sistema de Alertas:** Notificaciones automáticas de vencimiento (< 30 días), duplicidad de atención al paciente y alertas de seguridad de identidad.
* **Interfaz Moderna:** UI basada en **Glassmorphism** y Minimalismo, optimizada con Tailwind CSS para una experiencia de usuario fluida.

## 🛠️ Stack Tecnológico

* **Backend:** PHP 8.2 (Programación Orientada a Objetos).
* **Frontend:** Tailwind CSS, JavaScript (Vanila).
* **Base de Datos:** MySQL / MariaDB con PDO (PHP Data Objects).
* **Arquitectura:** Modelo-Vista-Controlador (MVC).
